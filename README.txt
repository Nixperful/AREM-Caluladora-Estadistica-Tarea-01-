Calculadora Estadística:

    1) Primero es necesario que clone el repositorio con git clone seguido del link de este.
    2) El archivo data.txt es el archivo de entrada de datos del programa, para ingresarlos de manera   correcta, es necesario que los divida con el salto de linea.
    3) Después de agregar los datos puede correr el programa con el siguiente comando:

        java -cp target/psp0-1.0-SNAPSHOT.jar calculadorestadistico.CalculadorEstadistico 

     Este finalmente le brindará los datos estadisticos de media y de desviación estandard.
